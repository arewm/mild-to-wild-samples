# Mild

**Is this attestation here? Is it valid?**

Start here. The mild level verifies that an artifact has SLSA provenance, that it was produced by a recognized build type and builder, and that its source materials and external parameters are properly recorded. These checks work regardless of build system because the attestation format itself is the interface.

See [conforma/](conforma/) and [ampel/](ampel/) for engine-specific policies and invocation instructions.
